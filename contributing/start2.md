### Done?
#### Run the test server
[Run the testing server](testingserver.md) and make sure it's all right.
#### `git add` what you want to commit
To commit all changes, do `git add *`.
#### `git commit`
Be sure to add a message using `-m`.
#### `git push`
This instantly uploads your changes.
